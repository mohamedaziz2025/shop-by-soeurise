# Logo Soeurise

Ce dossier contient tous les logos officiels de la marque Soeurise.

## Fichiers requis :

- `logo-main.png` - Logo principal (couleur)
- `logo-white.png` - Logo blanc (pour fond sombre)
- `favicon.ico` - Favicon du site

## Dimensions recommandées :

- Logo principal : 500x150px (format PNG avec transparence)
- Logo blanc : 500x150px (format PNG avec transparence)
- Favicon : 32x32px ou 16x16px (format ICO)

## Usage :

Les logos sont référencés dans le frontend Next.js via :
```tsx
<Image src="/logo-soeurise/logo-main.png" alt="Soeurise" width={150} height={50} />
```

Pour le moment, créez des logos temporaires ou utilisez des placeholders.
